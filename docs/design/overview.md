## デザイン原則

1. **シンプルさ** - 不要な装飾を避け、機能に集中
2. **一貫性** - 全体で統一されたデザイン言語
3. **アクセシビリティ** - すべてのユーザーに配慮
4. **レスポンシブ** - あらゆるデバイスで快適に
